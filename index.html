BeckerGPT – Fractal Prime Cryptocurrency

BeckerGPT is a mathematically grounded, symbolic cryptocurrency built upon the Becker–GPT Sieve: a prime-generation architecture based on cyclic residues (1, 3, 7, 9) and coprimes of 42.

Project Vision

This project aims to unite mathematics, cryptography, and symbolic intelligence in a real and functional cryptocurrency. All components are implemented in Python, including key generation using ECC, transaction signing and verification, mining, blockchain persistence, and a prototype P2P network node.
